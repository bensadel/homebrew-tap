# Instructions

## How do I install these formulae?

`brew install bensadel/tap/<formula>`

Or `brew tap bensadel/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bensadel/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
